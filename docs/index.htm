Package installation

Download the Sringy Editor package and upload the Springy folder in the root of your website.
After this step you can continue by reading how to or required file sections

Dependencies

Javascript enabled in the browser, jQuery and jQuery UI libraries are required.
Until now were not encountered issues with older versions of jquery nor with the recent ones, so you're free to use the jquery library more convenient for you.

Run in all modern browsers that supports browser commands.
Springy Editor also include Font-Awesome, available in the package.

Required file (inclusion)

To be able to run your Springy Editor instance you need to include some file (css and javascript) in the page.
In the page header, or wherever it is most convenient for you, add the following files in addition to the jQuery liraries:

  • /springy/res/font-awesome-4.3.0/css/font-awesome.min.css
  • /springy/css/editor.css
  • /springy/js/editor-connectors.js
  • /springy/js/editor.js

In this example I assume that the editor folder is located into the root of your website (as must be) <link type="text/css" href="/springy/res/font-awesome-4.3.0/css/font-awesome.min.css" rel="stylesheet" />
<link type="text/css" href="/springy/css/editor.css" rel="stylesheet" />

<script type="text/javascript" src="/springy/js/editor-connectors.js"></script>
<script type="text/javascript" src="/springy/js/editor.js"></script>

How to

Before start let's say that the editor can be loaded with a default text.
For example, you are in a scenario where you're going to edit a page or part of it and you need to preload the editor with your content.
For this purpose you can set the "field value" for Springy Editor, in the way your content can be loaded at start up.

To use Springy Editor in your page use the following instructions:

1) Add an iframe to your page and assign an id. In this example we'll use the id cfeditor <iframe src="" id="cfeditor"></iframe>

2) In your script (javascript) just use the following lines of code.
We'll use the hidden field id cfeditor_value var springyEditor = new Springy();
springyEditor.Create({ id: "cfeditor", fieldId: "cfeditor_value" });

SpringyConnectors

The connectors object is used for the editor components such as image browser, add link and color picker.
A connector allow the editor to open a module, called tool, in a modal dialog by getting it from an url.
Each connector (object) have it's configuration parameters as below:

url required the resource url where to get the html content for the tool
buttons required or null the modal dialog buttons (jquery). NOTE: set it to null if no buttons are required to override the base
postUrl this is used from form actions inside the tool (i.e.: image upload)

The connector items have the following configuration: 

myConnector: {
    url: "http://somewhere.go",
    postUrl: "action.ext"
    buttons: {
        "Ok": function () {
            ..code here...
        }
        "Cancel": function () {
            ..code here...
        }
    }
}

The connector configuration file is stored in the package at the path /Springy/js/editor-connectors.js
Change the configuration to match your needs.

Options

id The editor id
fieldId The field or input element id where to store the editor value
if you want set the start editor value (content) like when you're going to edit an existing content, you can fill the fieldId value and when the editor is ready it will set the content value, ready to be edited
width The editor width
height The editor height
autosize Indicate if the editor grow its heght based on the content. The default value is false
domain http://yourcooldomain.com/
usePageStyles Indicate if the editor font styles used in the page must be used in the editor. The default value is true
For a detailed list, go to page styles
connectors Of type SpringyConnectors
A connector allow the editor to open a module, such as image browser, add link, color picker, in a modal dialog by getting it from an url
style the editor toolbar style. The default value is blue.
Allowed value are blue,white,tone
blue provide a blue toolbar
white provide a white toolbar
tone provide a transparent background toolbar
